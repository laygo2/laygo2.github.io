Overview#

Introduction#

The LAYout with Gridded Object 2 (laygo2) package is a Python-based layout generation framework with the following goals:

  • Automation and reuse of layout generation process.

  • Parametrized layout generation.

  • Advanced technology support (such as FinFET).

  • (Dynamic) template-and-grid-based layout generation.

laygo2 extends the capability of its previous version (laygo), which is orignally developed as one of layout generation engines in Berkeley Analog Generator2 (BAG2).

Typical design flow#

  1. Load technology parameters, primitive templates, and grids from laygo2_tech.

    • Check laygo2_tech.load_templates() and laygo2_tech.load_grids().

  2. Generate instances from templates.

    • Check object.Template.generate().

  3. Place the generated instances.

    • Check object.database.Design.place().

  4. Route wires and vias between the instancesâ€™ terminals.

    • Check object.database.Design.route().

  5. Create Pins.

    • Check object.database.Design.pin().

  6. Export the generated design in proper format.

    • Check interface.skill.export().

  7. (Optional) export the design as a new template.

    • Check interface.yaml.export_template().
