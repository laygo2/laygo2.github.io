laygo2 - LAYout with Gridded Objects 2#

[Source Code] [Colab Tutorial]

The LAYout with Gridded Object 2 (laygo2) package is a Python package for automatic IC layout generation.

laygo2 is currently being maintained by the Nifty Chips Laboratory at Hanyang University, South Korea.
